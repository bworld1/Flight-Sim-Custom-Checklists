COCKPIT INITIALIZATION AND ENGINE START

 

 

IMPORTANT! EVEN WITH A FRESH PLANE, THERE IS ONLY ABOUT 5 MINUTES OF BATTERY POWER TO GET THE FIRST ENGINE STARTED, EVEN IF CART BATTERY POWER IS TURNED ON!

 

 

 

ON COPILOT PANEL, SWITCH ON “GROUND CREW” AND “BOARDING LADDER TRUCK” (RED LEVER AND BUTTON BELOW IT)

 

 

 

USE “SHIFT E” KEYS TO OPEN PASSENGER DOOR

 

 

 

VERIFY ALL ELECTRICS OFF - KEEP MASTER BATTERY SWITCH OFF UNTIL PREPARING TO START ENGINE

 

 

 

VERIFY ALL SYSTEMS ARE OFF

 

 

 

VERIFY ALL FLIGHT CONTROLS ARE OFF/DOWN

 

 

 

IN VIRTUAL COCKPIT OVERHEAD, VERIFY ALL MAGNETOS ARE OFF

 

 

 

IN VIRTUAL COCKPIT OVERHEAD, VERIFY LANDING LIGHTS OFF

 

 

 

IN VIRTUAL COCKPIT OVERHEAD, VERIFY NO SMOKING / SEATBELT SIGN SWITCHES ARE OFF

 

 

 

IN VIRTUAL COCKPIT VIEW, ON REAR OF CENTER CONSOLE, SET PARKING BRAKE DOWN / ON

 

 

 

NEXT TO PARKING BRAKE LEVER, VERIFY UNDERCARRIAGE GEAR LEVER IS DOWN AND LOCKED (INDICATORS IN THE CENTER MAIN PANEL SHOULD BE GREEN WHEN BATTERY IS ON)

 

 

 

VERIFY BOTH VOLTAGE GAUGES READ 24 VOLTS, YOU HAVE LESS THAN FIVE MINUTES TO START AN ENGINE!

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE ELCTRICAL PANEL, SELECT 2 INVERTER SWITCHES DOWN / ON

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE MAIN PANEL, SELECT ALL OIL COOLING FLAPS AND COWL FLAPS FULL OPEN, VERIFY ON COWL FLAP GAUGES

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE MAIN PANEL, SELECT ALL PROPELLER PITCH LEVERS FULL UP / HIGH RPM

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE MAIN PANEL, SELECT ALL FUEL TANK LEVERS AND SHUT OFF VALVES OPEN

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE MAIN PANEL, SELECT ALL MIXTURE LEVERS TO CUT OFF (LOWEST POSITION)

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE MAIN PANEL, SELECT THE MIXTURE LEVER FOR ENGINE 3 TO FULL RICH

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE ELECTRIC PANEL, SELECT ENGINE SELECTOR SWITCH TO NUMBER 3 (CYCLE THROUGH FROM #1)

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE ELCTRICAL PANEL, SELECT MASTER BATTERY SWITCH ON

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE ELECTRIC PANEL, TEST / SELECT STARTER SWITCH ON (WATCH FOR PROPELLER ROTATION IN EXTERNAL VIEW)

 

 

 

IN VIRTUAL COCKPIT VIEW, ON THE OVERHEAD CENTER PANEL, SELECT THE MAGNETO FOR ENGINE NUMBER 3 TO BOTH

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE MAIN PANEL, SELECT AUXILIARY FUEL PUMP FOR ENGINE 3 ON

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE ELECTRIC PANEL, SELECT PRIMER SWITCH TWICE TO DOWN / ON

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE MAIN PANEL, SET THE THROTTLE LEVER FOR ENGINE 3 UP TO ABOUT 6%

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE ELECTRIC PANEL, SELECT STARTER SWITCH ON (IF NOTHING HAPPENS, YOUR BATTERY HAS RUN OUT – BOTH VOLTAGE GAUGES WILL READ 20 VOLTS, YOU MUST RELOAD THE PLANE!)

 

 

 

WATCH ENGINE START UP IN EXTERNAL VIEW, MONITOR INSTRUMENTS AND WATCH FOR WARNING LIGHTS

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE ELCTRICAL PANEL, SELECT #3 GENERATOR SWITCHE UP / ON, VERIFY GAUGES SHOW AMPERAGE

 

 

 

IN FLIGHT ENGINEER VIEW, ON THE MAIN PANEL, SELECT AUXILIARY FUEL PUMP SWITCH FOR ENGINE 3 OFF

 

 

 

REPEAT PROCEDURE FOR ENGINES 4, 1, AND THEN 2

 

 

 

SELECT ENGINE SELECTOR SWITCH TO THE MIDDLE OR OFF POSITION

 

 

 

ONCE ALL ENGINES ARE STARTED AND RUNNING SMOOTHLY, RETURN THE THROTTLES TO IDLE AND VERIFY ALL GAUGES AND LIGHTS ARE IN NORMAL RANGE

 

 

 

SET UP NAVIGATION RADIOS AND AUTOPILOT

 

 

 

USE “SHIFT 2” KEYS TO DISPLAY RADIO STACK

 

 

 

ENTER FIRST VOR INTO NAV1, SECOND VOR INTO NAV2

 

 

 

USE “SHIFT 3” KEYS TO DISPLAY AUTOPILOT

 

 

 

ON PITCH PANEL, PRESS BOTTOM RIGHT BUTTON TO UNCAGE THE PITCH BARS

 

 

 

ON PITCH PANEL, SET TOP RIGHT KNOB FOR 3 DEGREES NOSE UP

 

 

 

ON HEADING PANEL, UPPER TAPE IS THE TARGET HEADING, SET UPPER RIGHT KNOB TO THE RUNWAY HEADING

 

 

 

ON HEADING PANEL, LOWER TAPE IS THE CURRENT HEADING, ALIGN TO THE CURRENT HEADING OF THE AIRCRAFT

 

 

 

TAXI

 

 

 

USE “SHIFT E” KEYS TO CLOSE PASSENGER DOOR IF OPEN

 

 

 

SWITCH OFF THE GROUND CREW AND GROUND LADDER CART

 

 

 

VERIFY NAV LIGHTS, LANDING LIGHTS ON

 

 

 

VERIFY ALTIMETER SET

 

 

 

ON BACK BULKHEAD, SET AIR CONDITIONER MASTER SWITCHES TO ON

 

 

 

VERIFY RADIO AND AVIONICS ON, AND AUTOPILOT IS SET/OFF

 

 

 

IN ENGINEER MAIN PANEL, SET COWL FLAPS TO 40%

 

 

 

IN VIRTUAL COCKPIT VIEW, SET ELEVATOR TRIM NOSE DOWN TO AROUND -7%

 

 

 

IN VIRTUAL COCKPIT, SET FLAPS TO TAKE-OFF POSITION

 

 

 

TAXI TO RUNWAY USING SMALL THROTTLE SETTINGS AND RUDDER TO STEER

 

 

 

TAKE OFF, CLIMB, AND CRUISE

 

 

 

AFTER ATC CLEARANCE, SWITCH ALL FUEL PUMPS ON

 

 

 

PERFORM RUN UP TEST – APPLY 1700 RPM OF POWER WITH THROTTLES

 

 

 

CONFIRM ALL INSTRUMENTS READ NORMAL

 

 

 

RETURN THROTTLES TO IDLE

 

 

 

APPLY 1700 RPM  OF POWER AGAIN AND TEST THE ALL PROPELLER PITCHES

 

 

 

RETURN ALL PROP PITCHES TO HIGH RPM, AND THROTTLES TO IDLE

 

 

 

FINAL CHECKLIST – VERIFY BRAKE SET, FUEL QUANTITY CORRECT, COWL FLAPS OPEN, THROTTLES IDLE, PROPS AT HIGH RPM, MIXTURES AT AUTORICH, ELEVATOR TRIM SET, FLAPS AT ONE NOTCH, FLIGHT CONTROLS CHECK, RADIOS/AVIONICS SET, LANDING LIGHTS ON, CARBURETORS ARE COLD

 

 

 

OPEN THROTTLES TO MAXIMUM

 

 

 

RELEASE BRAKES

 

 

 

ACCELERATE THROUGH V1 – 90 KNOTS

 

 

 

ROTATE AT VR – 100-110 KNOTS

 

 

 

TAKE OFF SLOWLY – NOSE WHEEL WILL UNSTICK, SLOWLY LEAVE RUNWAY TO CLEAR TAIL, THEN PULL BACK TO UNSTICK MAIN GEAR

 

 

 

APPY NOSE DOWN PRESSURE TO LET AIR SPEED INCREASE

 

 

 

RAISE GEAR, VERIFY THAT TAIL SKID HAS RETRACTED IN EXTERNAL VIEW, AND THREE UNDERCARRIAGE LIGHTS ARE RED

 

 

 

CLIMB AT 500 FPM AT 142 KNOTS

 

 

 

ADJUST POWER / MANIFOLD PRESSURE TO 42” AND PROP PITCH TO 2300 RPM

 

 

 

AT 1000 FT., ADJUST MANIFOLD PRESSURE TO 35” AND KEEP PROP PITCH AT 2300 RPM

 

 

 

AT 1500 FT., RAISE FLAPS AND SET AUTOPILOT ON

 

 

 

AT 9000 FT., ADJUST PROP PITCH FOR 152 KNOTS AND CLOSE COWL FLAPS TO 30%

 

 

 

ABOVE 9000 FT., ADJUST MANIFOLD PRESSURE TO 33” AND KEEP PROP PITCH AT 2300 RPM FOR 152 KNOTS

 

 

 

AT 21000 FT., ADJUST MIXTURE AND PROP PITCH FOR 142 KNOTS

 

 

 

AT CRUISE ALTITUDE, LEVEL OFF, SET AUTOPILOT PITCH FOR LEVEL FLIGHT, SET COWL FLAPS TO OPTIMUM TEMPERATURE, SET MANIFILD PRESSURE TO 31”, SET RPM TO 2300

 

 

 

FOLLOW VOR HEADINGS

 

 

 

WHEN ATC COMMANDS RESUME OWN NAVIGATION, TUNE THE COURSE KNOB UNTIL THE CDI CENTERS

 

 

 

ADJUST AUTOPILOT HEADING TO COURSE HEADING TO FLY TO FIRST VOR

 

 

 

AT CROSSING VOR, SWITCH TO NAV2 FOR VOR2, REPEAT ABOVE STEPS

 

 

 

SET NAV1 FOR THIRD VOR, ETC, UNTIL APPROACHING DESTINATION

 

 

 

APPROACH DESCENT CHECKLIST

 

 

 

ABOVE 20,000 FT., REDUCE SPEED TO 210 KIAS

 

 

 

BELOW 20,000FT., DO NOT EXCEED 236 KIAS

 

 

 

ADJUST MIXTURES TO AUTORICH

 

 

 

OPEN COWL FLAPS TO 10%

 

 

 

MAINTAIN 2000 RPM BY GRADUALLY REDUCING MANIFOLD PRESSURE AND RPM

 

 

 

AT 10,000 FT., SET LANDING LIGHTS ON

 

 

 

ON APPROACH, REDUCE AIRSPEED TO 152 KIAS

 

 

 

LOWER FLAPS ONE NOTCH

 

 

 

APPROACH GLIDESLOPE, REDUCE SPEED TO 125 KIAS, LOWER FLAPS ONE NOTCH

 

 

 

ON ATTAINING GLIDESLOPE, SET AIRSPEED TO 110 KIAS, AND SET FLAPS TO FULL

 

 

 

LOWER GEAR AND VERIFY THREE GREEN LIGHTS

 

 

 

ADJUST ELEVATOR TRIM

 

 

 

LANDING CHECKLIST

 

 

 

AT CROSSING RUNWAY, REDUCE AIRSPEED TO 90-95 KIAS AND BEGIN FLARE

 

 

 

BEFORE TOUCHDOWN, CLOSE THROTTLES AND SETTLE ON MAIN GEAR FIRST

 

 

 

GENTLY LOWER NOSE GEAR

 

 

 

RAISE FLAPS AND OPEN COWL FLAPS FULL

 

 

 

PARKING CHECKLIST

 

 

 

STOP NEAR GATE, SET PARKING BRAKE ON

 

 

 

PULL THROTTLES DOWN TO IDLE

 

 

 

PRESS “CONTROL .” TO SET PARKING BRAKE ON

 

 

 

ON OVERHEAD, SELECT PASSENGER SIGNS OFF, LANDING LIGHTS OFF

 

 

 

 ENABLE LADDER TRUCK

 

 

 

PRESS “SHIFT E” TO OPEN MAIN DOOR

 

 

 

CLOSE EACH MIXTURE LEVER  TO CUT ENGINES – 2, 1, 4, 3

 

 

 

SET PROP PITCHES TO MINIMUM

 

 

 

CLOSE ALL FUEL PUMPS AND FUEL TANKS

 

 

 

SWITCH CART BATTERY ON

 

 

 

SWITCH MASTER BATTERY OFF

 

 

 

SET ALL INVERTERS AND GENERATORS OFF

 

 

 

ALL LIGHTS OFF
